Podcast sources

Edits write directly to config/sources.yaml and take effect on the next daily run. Disabled sources are kept in the file but skipped at runtime.

Ad-hoc summary

Paste a YouTube URL or a podcast URL (direct audio file, RSS feed, or an episode page that links the audio) to summarize one episode right now. The brief is emailed immediately; the episode also rolls up into tomorrow's daily roundup. YouTube takes 1–3 min; podcasts transcribe on a GPU and can take 10–15 min (don't navigate away).
