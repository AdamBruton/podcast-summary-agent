Ad-hoc summary

Paste a YouTube URL to summarize one episode right now. The brief is emailed immediately; the episode also rolls up into tomorrow's daily roundup. Typical processing time: 1–3 minutes (don't navigate away).

Individuals & companies

Discovery searches YouTube daily for each of these names — people or companies — curates results via Claude, and routes approvals into the brief pipeline.

    Database backup & restore

    Daily backups run automatically before each scheduled brief and rotate to the last 14 snapshots on the volume. A weekly off-site copy is emailed via SendGrid on Sundays. Restoring snapshots the current state.db, then swaps in the uploaded file in place — no service restart required.

    Restore from file

    Select an uncompressed state.db file. If you have a .db.gz snapshot, gunzip it first.
